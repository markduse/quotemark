## Why IUL is hard (read this section before coding)

IUL is NOT a flat-rate quote like term/FE. It's a **projected savings vehicle** with:

- **Premium** — agent + client choose what to fund monthly
- **Death benefit option** — Option A (level) vs Option B (DB rises with cash value)
- **Cash value accumulation** — driven by an index (usually S&P 500) with:
  - **Cap rate**: max annual gain credited
  - **Participation rate**: % of index gain credited
  - **Floor**: minimum credit (usually 0%)
  - **Spread**: deduction from index
- **Loads + COI (cost of insurance)** — deducted from cash value monthly
- **Surrender charges** — declining over ~10 years
- **Illustrated rate**: AG49-compliant standard (currently 6.0%)

A real IUL **illustration** (the PDF agents show clients) is generated by carrier software with Monte Carlo or scenario-based projections. **QuoteMark cannot reproduce these exactly without replicating each carrier's illustration engine** — which is essentially impossible without proprietary access.

## Three scope options — pick before coding

### Scope A — "Premium for target DB" comparator (RECOMMENDED)

Show, for a given (age, gender, tobacco, face amount, premium duration):

| Carrier | Required Monthly Premium | Cap | Par | Floor | Projected CV @ Age 65 / 75 / 85 |

Required premium is calculable from each carrier's "minimum funding" tables. Cash value projections use AG49 illustrated rate (6.0%) × the carrier's strategy parameters.

**Effort:** 4-6 hours total for 4 carriers.
**Accuracy bar:** within 5% of carrier illustration software (good enough for agent decision-making, not for client deliverables).

### Scope B — "Carrier feature comparison" (FASTEST)

No quoting at all. Static comparison table:

| Carrier | Issue Ages | Face Range | Cap | Par | Floor | DB Options | Riders |

Informational, helps agents pick which carrier to illustrate (then they use the carrier's own software for the actual client illustration).

**Effort:** 1-2 hours.
**Use case:** "Which carrier should I quote for this client?" → leads to carrier illustration software for the actual sale.

### Scope C — Full illustration engine (DON'T DO)

Reproduce carrier illustration PDFs in-app. Don't attempt. Compliance risk, complexity, zero value over carrier software.
